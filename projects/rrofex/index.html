Conector rRofex

rRofex es una librería de R que permite conectarse a Mercado de Capitales Argentino.

Es un proyecto OpenSource desarrollado desde Primary, empresa de tecnología de Matba Rofex.

Esta librería se encuentra publicada oficialmente en CRAN y su versión de desarrollo está en GitHub

Para mayor información sobre la misma visitá rRofex.

Ver a la presentación!

Project link: https://matbarofex.github.io/rRofex
